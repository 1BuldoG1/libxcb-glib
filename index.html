libxcb-glib

A library to integrate XCB nicely with GLib

libxcb-glib provides integration of XCB events to the GLib main loop.

Dependencies

  • XCB
  • GLib

Releases download

Visit the downloads page to find releases (tar.xz) and automatic GitHub tag tarball/zip.

Git repository

The repository is hosted on GitHub: libxcb-glib Git repository

You can clone the project with Git by running:
$ git clone git://github.com/sardemff7/libxcb-glib
